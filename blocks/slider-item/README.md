# Slider Item Block

A container block specifically designed to be used inside the MBN Slider block. Each slider item can contain any WordPress blocks.

## Features

- **Nested Block Support** - Add any WordPress blocks inside
- **Parent Restriction** - Can only be used inside MBN Slider block
- **Min Height Control** - Set custom minimum height for each item
- **Visual Indicator** - Shows "Slider Item" label in editor
- **Flexible Content** - Add images, text, buttons, or any blocks

## Usage

This block is automatically added when you use the MBN Slider block. You cannot add it directly - it only appears inside MBN Slider.

### In Editor

1. **Add MBN Slider** block to your page
2. **Three Slider Items** are created automatically
3. **Click inside any item** to add blocks
4. **Add more items** by clicking the + button in the slider
5. Each item shows "Slider Item" label in the top-left corner

### Item Settings

Each slider item has its own settings in the sidebar:

- **Min Height** - Set minimum height (e.g., 400px, 50vh)

## Technical Details

**Block Name**: `mbn-theme/slider-item`  
**Parent Block**: `mbn-theme/mbn-slider` (required)  
**Category**: Layout  
**Reusable**: No (designed for single use within slider)

## File Structure

```
blocks/slider-item/
├── block.json          # Block metadata with parent restriction
├── index.js            # Registration
├── edit.js             # Editor with InnerBlocks
├── save.js             # Frontend output
└── style.css           # Item styles
```

## Example Content

**Testimonial Item:**
```
- Paragraph (quote)
- Heading (author name)
- Image (avatar)
```

**Feature Card Item:**
```
- Image
- Heading (title)
- Paragraph (description)
- Button (CTA)
```

**Hero Slide Item:**
```
- Cover block (background image)
- Heading (hero title)
- Paragraph (description)
- Buttons (CTAs)
```

## Styling

The slider-item block provides minimal styling to allow maximum flexibility:

- Full width and height
- Basic spacing for child elements
- Visual editor indicators (dashed border, label)
- Responsive behavior inherited from MBN Slider

## Notes

- This block cannot be used outside of MBN Slider
- Not available in the block inserter directly
- Automatically generated by MBN Slider template
- Min height can be set per item or globally in MBN Slider settings
